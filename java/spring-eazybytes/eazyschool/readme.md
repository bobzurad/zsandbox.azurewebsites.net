This project is created from the following Udemy Course: [Master Spring, Spring Boot, REST, JPA, Hibernate](https://www.udemy.com/course/spring-springboot-jpa-hibernate-zero-to-master) (Section 6, and onwards)

## Prerequisites

This project requires the following
* Java 17
* Maven 3

Note: This project contains a `.tool-versions` file. So if you have [asdf](https://asdf-vm.com) installed, you can run `asdf install java` to ensure that the proper version of Java is installed.

## Building

To build the project, run: `mvn clean package`

## Running

To run the project, run: `mvn spring-boot:run`
